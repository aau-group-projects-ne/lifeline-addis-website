Welcome to the lifeline addis project!

To collaborate in this project,
First fork this repository into your local repo.
Afterwards, clone the repo and create a branch under your name to work on.

To set up the backend, the password and username of the local user of the databse is already written in ./src/models/db.js, so change it according to your local sql database details.

as for the tables, the tables used are already in dbFile.sql, so use those table queries to set up the tables in your local database.

!!Warning: Work on your own named branches and pull to your main branch whenever there are merges done to the main branch of this project to be upto date.

Happy Coding!
